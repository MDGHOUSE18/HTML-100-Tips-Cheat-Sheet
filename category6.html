61. Table Caption 
<table>
  <caption>Monthly Sales</caption>
  <tr><th>Month</th><th>Sales</th></tr>
  <tr><td>January</td><td>$500</td></tr>
</table>

When/Why: Adds a title/description for a table, improving accessibility.

62. Thead, Tbody, Tfoot 
<table>
  <thead>
    <tr><th>Product</th><th>Price</th></tr>
  </thead>
  <tbody>
    <tr><td>Apple</td><td>$1</td></tr>
  </tbody>
  <tfoot>
    <tr><td>Total</td><td>$1</td></tr>
  </tfoot>
</table>

When/Why: Separates table header, body, and footer for clarity and accessibility.

63. Column Headers with Scope="col" 
<th scope="col">Name</th>

When/Why: Defines column headers for screen readers.

64. Row Headers with Scope="row" 
<tr><th scope="row">Apples</th><td>$2</td></tr>

When/Why: Defines row headers to help with accessibility.

65. Colspan 
<td colspan="2">Merged Cell</td>

When/Why: Merges multiple columns into one cell.

66. Rowspan 
<td rowspan="2">Merged Row</td>

When/Why: Merges multiple rows into one cell.

67. Colgroup & Col Styling 
<table>
  <colgroup>
    <col style="background:#f0f0f0;">
    <col style="background:#e0f7fa;">
  </colgroup>
  <tr><td>A</td><td>B</td></tr>
</table>

When/Why: Styles entire columns at once.

68. Responsive Tables with Overflow 
<div style="overflow-x:auto;">
  <table>
    <tr><th>Item</th><th>Details</th></tr>
    <tr><td>Laptop</td><td>15-inch</td></tr>
  </table>
</div>

When/Why: Prevents tables from breaking layouts on small screens.

69. ARIA Grid Role 
<table role="grid">
  <tr><td>Cell 1</td><td>Cell 2</td></tr>
</table>

When/Why: Helps assistive tech understand table layout as a grid.

70. ARIA Sort Attribute 
<th aria-sort="ascending">Name</th>

When/Why: Indicates if a column is sorted (ascending/descending) for accessibility.
