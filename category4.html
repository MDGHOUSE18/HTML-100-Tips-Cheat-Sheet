41. Alt Attribute 
<img src="dog.jpg" alt="A happy brown dog">

When/Why: Provides alternative text for accessibility and SEO. Essential for screen readers.

42. Title Attribute on Images 
<img src="sunset.jpg" alt="Sunset" title="Beautiful sunset at the beach">

When/Why: Adds a tooltip on hover with extra description.

43. Srcset for Responsive Images 
<img src="small.jpg" 
     srcset="medium.jpg 600w, large.jpg 1200w" 
     alt="Mountains">

When/Why: Serves different image sizes based on screen width for better performance.

44. Sizes with Srcset 
<img src="default.jpg" 
     srcset="img-480.jpg 480w, img-800.jpg 800w" 
     sizes="(max-width: 600px) 480px, 800px" 
     alt="Responsive image">

When/Why: Helps the browser pick the right image size depending on device resolution.

45. Picture Element 
<picture>
  <source srcset="image.webp" type="image/webp">
  <source srcset="image.jpg" type="image/jpeg">
  <img src="fallback.jpg" alt="Scenic view">
</picture>

When/Why: Lets you serve different image formats (WebP, JPG) for performance.

46. Lazy Loading 
<img src="big-photo.jpg" alt="Landscape" loading="lazy">

When/Why: Defers loading off-screen images until needed, improving page speed.

47. Decoding Async 
<img src="photo.jpg" alt="Fast loading" decoding="async">

When/Why: Improves performance by letting the browser decode images asynchronously.

48. Width & Height Attributes 
<img src="banner.jpg" width="600" height="300" alt="Banner">

When/Why: Prevents layout shifts by reserving space for images before they load.

49. Figure & Figcaption 
<figure>
  <img src="art.jpg" alt="Artwork">
  <figcaption>Modern abstract painting</figcaption>
</figure>

When/Why: Groups images with captions for semantic meaning.

50. Inline SVG 
<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="blue" stroke-width="3" fill="lightblue" />
</svg>

When/Why: Scales without quality loss and allows styling/animation with CSS.
