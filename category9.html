91. Meta Description 
<meta name="description" content="Learn 100 HTML tips and tricks.">

When/Why: Helps search engines display meaningful snippets in search results.

92. Viewport for Mobile 
<meta name="viewport" content="width=device-width, initial-scale=1.0">

When/Why: Ensures your site scales correctly on mobile devices.

93. Canonical URL 
<link rel="canonical" href="https://example.com/page">

When/Why: Avoids duplicate content issues for SEO.

94. Preload Critical CSS 
<link rel="preload" href="styles.css" as="style">

When/Why: Loads important CSS earlier to improve performance.

95. DNS Prefetch 
<link rel="dns-prefetch" href="//cdn.example.com">

When/Why: Resolves DNS early for external resources.

96. Preconnect 
<link rel="preconnect" href="https://fonts.googleapis.com" crossorigin>

When/Why: Establishes a quick connection for faster external requests.

97. Open Graph Meta Tags 
<meta property="og:title" content="100 HTML Tips">
<meta property="og:description" content="Master HTML with these tips.">
<meta property="og:image" content="thumbnail.jpg">

When/Why: Improves social media sharing previews.

98. Twitter Card Meta Tags 
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="100 HTML Tips">
<meta name="twitter:description" content="HTML tips for SEO and performance.">

When/Why: Optimizes how your content looks on Twitter.

99. HTML Language Attribute 
<html lang="en">

When/Why: Helps screen readers and search engines understand the page language.

100. Robots Meta 
<meta name="robots" content="index, follow">

When/Why: Tells search engines whether to index and follow links.
