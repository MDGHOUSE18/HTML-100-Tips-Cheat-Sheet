51. Open in New Tab (Secure) 
<a href="https://example.com" target="_blank" rel="noopener">Visit Example</a>

When/Why: Opens link in a new tab while preventing security risks from window.opener.

52. Download Attribute 
<a href="report.pdf" download>Download Report</a>

When/Why: Forces file download instead of opening it in the browser.

53. Nofollow Links 
<a href="https://example.com" rel="nofollow">External Link</a>

When/Why: Tells search engines not to pass SEO authority to the linked page.

54. Noopener Noreferrer 
<a href="https://example.com" target="_blank" rel="noopener noreferrer">Safe Link</a>

When/Why: Improves security and privacy for external links.

55. Anchor IDs (Internal Links) 
<a href="#section1">Go to Section 1</a>
<h2 id="section1">Section 1</h2>

When/Why: Jumps to specific parts of the same page, great for long content.

56. Skip Links (Accessibility) 
<a href="#main">Skip to Content</a>
<main id="main">Main content here</main>

When/Why: Lets screen reader and keyboard users skip repetitive menus.

57. Accesskey Attribute 
<a href="/help" accesskey="h">Help</a>

When/Why: Creates keyboard shortcuts for navigation links.

58. Navigation Role 
<nav role="navigation">
  <a href="/">Home</a>
  <a href="/about">About</a>
</nav>

When/Why: Defines navigation region for assistive technologies.

59. Breadcrumb Navigation 
<nav aria-label="breadcrumb">
  <ol>
    <li><a href="/">Home</a></li>
    <li><a href="/articles">Articles</a></li>
    <li>Current Article</li>
  </ol>
</nav>

When/Why: Improves navigation and SEO, especially for deep sites.

60. Link Prefetching 
<link rel="prefetch" href="/next-page.html">

When/Why: Loads resources in advance, making navigation faster.
