HTML Advanced Attributes (Tips 81–90)

81. Contenteditable 
<div contenteditable="true">Edit this text directly!</div>

When/Why: Makes any element editable like a text field.

82. Spellcheck 
<textarea spellcheck="true">Check my spelling</textarea>

When/Why: Enables or disables spell checking in inputs and textareas.

83. Draggable 
<img src="logo.png" draggable="true" alt="Logo">

When/Why: Makes an element draggable with native drag-and-drop API.

84. Translate Attribute 
<p translate="no">BrandName</p>

When/Why: Prevents specific text from being translated by auto-translators.

85. Hidden Attribute 
<div hidden>You can’t see me!</div>

When/Why: Hides elements without using CSS (still present in DOM).

86. Autofocus 
<input type="text" autofocus placeholder="I am focused on load">

When/Why: Automatically sets focus to an input when the page loads.

87. Autocomplete 
<input type="email" autocomplete="on" placeholder="Enter your email">

When/Why: Enables browser autofill for faster form completion.

88. Input Pattern 
<input type="text" pattern="[0-9]{3}-[0-9]{2}-[0-9]{4}" placeholder="123-45-6789">

When/Why: Enforces input format using regex without JavaScript.

89. Sandbox Iframe 
<iframe src="demo.html" sandbox></iframe>

When/Why: Restricts iframe content for better security (no scripts, forms, etc. by default).

90. Srcdoc in Iframe 
<iframe srcdoc="<p>Hello from inside an iframe!</p>"></iframe>

When/Why: Embeds inline HTML directly into an iframe without external files.
